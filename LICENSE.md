## 🎯 **عن المشروع**

**مشروع Djomoa** هو محرك حسابي متطور يعتمد على الثوابت الفلكية القطعية لتقديم تزامن زمني لحظي ودقيق بين التقاويم الشمسية والقمرية.

النظام مصمم ليكون **"العمود الفقري"** (Core Engine) لأي منصة رقمية تتطلب دقة تاريخية لا تقبل الخطأ، متجاوزاً بذلك الحلول الحالية التي تعتمد على التقديرات الإحصائية.

### ﴿ لِتَعْلَمُوا عَدَدَ السِّنِينَ وَالْحِسَابَ ﴾

## 🏛️ **الرؤية والأهداف**

### الرؤية
تقديم المرجعية الزمنية الأكثر دقة في التاريخ الرقمي، وفتح آفاق جديدة لم تكن متاحة لأي نظام تقويمي أو فلكي موجود حالياً.

### الأهداف الاستراتيجية
1. **توحيد الزمن**: تقديم معيار تقني لتوحيد وتوثيق الزمن في التطبيقات العالمية
2. **الدقة المطلقة**: تحقيق تزامن مثالي بين التقويمين الشمسي والقمري
3. **الاستقلالية**: نظام مغلق لا يعتمد على خوادم خارجية
4. **القابلية للتوسع**: جاهز للدمج مع أنظمة الذكاء الاصطناعي الضخمة
5. **المرجعية**: أن يكون المرجع الزمني للأنظمة الرقمية في مجالات التأريخ والبحوث الفلكية

📝 الاستخدام

تهيئة المحرك

```javascript
import { MasterEngine } from './Master_Engine.js';

// إنشاء محرك جديد
const engine = new MasterEngine();

// حساب 50,000 سنة
const result = engine.calc(50000);
console.log('نتيجة الحساب:', result);
```

تشغيل العداد

```javascript
// من Console المتصفح
window.restartCounter();    // إعادة تشغيل العداد من الصفر
window.showEngineInfo();    // عرض معلومات المحرك في Console
```

دوال JavaScript المتاحة

الدالة الوصف الاستخدام
initializeEngine() تهيئة المحرك الرئيسي await initializeEngine()
startCounter() تشغيل العداد المتحرك startCounter(element, target, duration)
main() الدالة الرئيسية للتشغيل main()
restartCounter() إعادة تشغيل العداد window.restartCounter()
showEngineInfo() عرض معلومات المحرك window.showEngineInfo()

أمثلة إضافية

```javascript
// تخصيص العداد
const counter = document.getElementById('syncCounter');
startCounter(counter, 1000000, 3000); // مليون خلال 3 ثوان

// إيقاف العداد
const stopCounter = startCounter(counter);
// ... لاحقاً
stopCounter(); // إيقاف العداد
```
